# Sales Channel Performance Analytics Dashboard

This project analyzes sales performance using the AdventureWorksDW2022 data warehouse.  
The goal is to build a complete business analytics portfolio project covering SQL data exploration, data cleaning, Power BI data modeling, DAX measures, dashboard design, and business insights.

## Tools

- SQL Server
- SQL Server Management Studio
- Power BI Desktop
- DAX
- GitHub
